# Звуковые файлы для игры

Сюда положите фоновую музыку и при необходимости другие звуки.

## Фоновая музыка в сценариях и мини-игре

- **Файл по умолчанию:** `game-bg.mp3`
- Положите в эту папку: `frontend/public/music/game-bg.mp3`
- Музыка проигрывается в цикле во время прохождения сценариев (Накопления, День рождения, Квиз, Бизнес, Инвестиции) и в мини-игре «Остров сокровищ».
- Включение/выключение: **Профиль → раздел «Профиль» → чекбокс «Музыка в сценариях и мини-игре»** (по умолчанию включено).

## Другой файл или имя

Чтобы использовать другой файл (например `ambient.ogg`), измените константу `MUSIC_SRC` в файле  
`frontend/src/contexts/MusicContext.jsx`:

```js
const MUSIC_SRC = '/music/ambient.ogg';
```

Файлы из `public/` доступны по корневому пути: `/music/game-bg.mp3`, `/music/ambient.ogg` и т.д.

## Рекомендации

- Формат: MP3 или OGG (хорошая совместимость с браузерами).
- Длина: 1–3 минуты (зацикливание).
- Громкость в коде задаётся на 50% (`audio.volume = 0.5`), при необходимости измените в `MusicContext.jsx`.
